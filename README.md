# 🍔 BiteRush

BiteRush is a modern food ordering web application inspired by platforms like Zomato and Swiggy, developed using **HTML, CSS, and JavaScript**. The project delivers a smooth, responsive, and interactive ordering experience while showcasing advanced front-end development concepts.

## Features

* Restaurant listing with dedicated menu pages
* Interactive food browsing experience
* Smart search functionality for restaurants and menu items
* User Sign Up and Login interface
* Dynamic shopping cart with real-time updates
* Increase/decrease item quantities instantly
* Automatic bill calculation with GST and additional charges
* Live total amount updates without page refresh
* QR code-based payment interface for checkout
* Fully responsive design across desktop, tablet, and mobile devices

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## Highlights

* Responsive and user-centric interface
* Real-time DOM manipulation and state updates
* Clean component-based folder organization
* Interactive cart and billing system
* Performance-focused front-end implementation
* Modern UI inspired by industry-leading food delivery platforms

This project was built to strengthen practical front-end development skills by implementing real-world e-commerce functionality using core web technologies.
